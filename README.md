# BirthdayBot
 All other birthday bots are either barely working or ask money for stupid features. I got tired of it so I made my own.


 ### Docker Environment Params:
 ```
 -e SQLSERVER=<MariaDB/MySQL host> 
 -e PORT=3306 
 -e USER=<db user>
 -e PASSWORD=<password>
 -e DATABASE=<db name>
 -e TOKEN=<discord bot token>
 ```
